■ 구버전 flash 삭제

저작권 때문에 파일을 직접올리지 못하니 링크(어도비)를 따라가서 직접 다운받거나 실행시키주세요^^

(저작권을 준수하는 문자노트;)

http://helpx.adobe.com/flash-player/kb/uninstall-flash-player-windows.html

 

 

 

위의 링크에 접속하시면 위와 같이 어도비 도움센타에 접속이 되는데 아랫쪽 빨간박스의 uninstaller를

다운받아서 실행하시거나 바로 실행하시면 됩니다.

 

이제 제거 버튼을 눌러 구버전의 flash를 깨끗히 제거 하시면 됩니다.

다른 방법으론 아무리 해도 안지워 졌는데 이방법은 아주 확실하게 제거가 됩니다...
