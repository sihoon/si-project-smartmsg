단문&장문 메시지 웹 연동

내 홈페이지나 쇼핑몰에서 자동으로 메시지를 발송하고 싶습니다.

  1. 1. 메시지를 발송할 페이지에 아래 스크립트를 복사하여 페이지에 넣어 주세요.

  2. 2. '발급받은 코드', '메시지', '전화번호', '보내는사람 번호'를 입력 하세요.(끝!)

주의사항

  • 해당 페이지가 모두 로딩된 이후에 메시지가 발송됩니다.(로딩전 페이지 이동시 발송되지 않을수 있습니다.- 로딩 후 발송완료 되었는지 확인 하는법)
  • 해당 페이지가 접속 될때마다 발송 됩니다.
내 홈페이지나 쇼핑몰에서 메시지를 직접 입력하여 발송하고 싶습니다.(Demo)

  1. 1. 발송 핸드폰이 보일 페이지 하단에 아래 css와 스크립트를 복사하여 페이지에 넣어 주세요.

  2. 2. '발급받은 코드', '핸드폰이 보여질 태그 아이디'를 입력 하세요.(끝!)

    3. '핸드폰이 보여질 태그 아이디'는 핸드폰이 보여질 태그로써 해당 태그내에 핸드폰이 보여집니다.(예. <div id="myID"><div>)

참고

  • 90byte 입력시 단문, 그 이상시 장문으로 발송 됩니다.
  • 장문을 못보내게 하려면? 위 코드중 // 부분을 지워 주세요.
